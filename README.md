A simple **GUI-based Bank Management System** developed using **Core Java** and **Java Swing**.  
The application allows users to perform basic banking operations such as account creation, deposits, withdrawals, and balance enquiry through an easy-to-use graphical interface.

---

## Features
- Create new bank accounts  
- Deposit money into an account  
- Withdraw money from an account  
- Check account balance  
- User-friendly GUI using Java Swing

  - Java Collections (`HashMap`)  
  - Event Handling  
  - Exception Handling
